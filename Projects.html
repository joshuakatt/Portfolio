3D Rubiks Cube Visualizer/Solver




As a child, I always found Rubiks Cubes intriguing.
The seemingly infinite faces a cube could have, the directions in which it could be twisted, and of course, the ways in which one could solve the cube.

I developed this 3D visualization program using Python and Matplotlib (which I happened to use in one of my courses.)
You can move the cube around in 3D, rotate faces of the cube and undo your steps. Eventually, I plan to make this a software to teach people on how to solve a cube.
I plan to implement this by Guiding the user in the right steps and warning the user when wrong steps are being made.

You can find the code to this Project here: Github

I also made a version of it with a Pyraminx (tetrahedron-shaped puzzle with four triangular faces), let me know if you're interested in seeing this!
